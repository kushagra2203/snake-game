# 🐍 Snake Game (Python)

A classic **Snake Game** built using Python. Control the snake, eat food, grow longer, and try to beat your high score!

---

## 🚀 Features

* 🎮 Smooth snake movement
* 🍎 Food generation system
* 📈 Score tracking + high score saved (`data.txt`)
* 💥 Collision detection (wall & self)
* 🔁 Game restart functionality

---

## 📂 Project Structure

```
snake-game/
│── snake.py                 # Snake movement logic
│── food.py                  # Food spawning
│── scoreboard.py            # Score tracking
│── data.txt                 # Stores high score
│── day_20 (snake game).py   # Main game file
```

---

## 🛠️ Requirements

Make sure Python is installed.

Install required library:

```
pip install turtle
```

---

## ▶️ How to Run

```
python "day_20 (snake game).py"
```

---

## 🎯 Controls

| Key | Action     |
| --- | ---------- |
| ↑   | Move Up    |
| ↓   | Move Down  |
| ←   | Move Left  |
| →   | Move Right |

---

## 📊 Game Logic

* Eat food → score increases
* Snake grows longer
* Game ends if:

  * Snake hits wall
  * Snake hits itself
